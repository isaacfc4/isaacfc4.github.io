Projects

Live Cell Imaging Environment

The objective of this project was to design and produce a device that provides a temperature regulated environment for the imaging of live-cells in a 96 well plate. Our team underwent a PDR, CDR, and formal engineering report to provide progress to client. The platform utilized an Arduino Uno and small electronic components to create a thermistor-based control system as well as 3D modeling, GibbsCAM, and shop machinery to fabricate product components.

View Project

Virtual Maze with Cozmo AI

The objective of this personal project was to simulate a physical environment inside the memory of the Cozmo robot. The software uses libraries provided by the Cozmo SDK to create a virtual environment modeled after a cardboard maze. The Cozmo platform would then treat the virtual structure as a 'barrier' and calculate the nominal path to a given coordinate and traverse the maze accordingly. This project utilized hundreds of lines of Python and acts as a transition project for young Python students looking to learn about the applications of python in machine learning.
